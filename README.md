# FDE Support Assistant API

FastAPI 기반 고객지원 AI Assistant API입니다.

## Goal

- 고객 문의를 입력받아 AI 응답을 생성합니다.
- 문의 카테고리와 심각도를 분류합니다.
- API 응답 시간과 처리 결과를 로깅합니다.
- Docker로 실행 가능한 백엔드 형태를 목표로 합니다.

## Tech Stack

- Python 3.12
- FastAPI
- OpenAI API
- Docker
- pytest

## Week 1 Scope

- FastAPI API 서버 구현
- Support chat endpoint 구현
- LLM 연동
- 기본 예외 처리
- Docker 실행 환경 구성
